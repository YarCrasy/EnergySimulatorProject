## Resumen
- Componentes UI reutilizables para layout, tarjetas y utilidades.

## Contenido
- Header/Footer/Logo: navegacion global.
- PrivateRoute: guardia de rutas.
- Cards y forms: `ProjectCard`, `CardPanel`, `RegisterForm`.
- Utilidades: `Spiner`, `SearchBar`, `ForceOrientation`, `UserProfile`.
- Admin: subcarpeta con tablas, formularios y cards.

## Diseno UI
- Estilo glassmorphism con gradientes y bordes redondeados.
