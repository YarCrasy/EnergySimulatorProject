## Resumen
- Tarjeta compacta para elementos del catalogo.

## Que hace
- Muestra imagen, modelo y potencia formateada.
- Soporta modo draggable para drag-and-drop.

## Diseno UI
- Card horizontal con media cuadrada y textos en columna.

## Props
- `imgSrc`, `id`, `model`, `wattage`, `draggable`, `onDragStart`, `onClick`.
