## Resumen
- Componentes del panel admin (usuarios y receivers).

## Contenido
- `cardReceiver/ReceiverCard`: tarjeta de receiver con acciones.
- `formReceiver/FormReceiver`: modal para crear/editar receivers.
- `navBar/NavBar`: barra de navegacion admin.
- `tableUsers/TableUsers`: tabla de usuarios con acciones.
- `formPanel/FormPanel`: formulario base de paneles (actualmente standalone).

## Diseno UI
- Predomina el estilo de cards con sombras y botones de accion.
